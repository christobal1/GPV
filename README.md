# GPV

    Workout progress tracker.
    Can be used for every kind of tracking and representing numbers.

    Instructions can be found in the program itself.

    (Currently locked at 50 weeks of tracking, can be changed in the code easily).

## Features

- Visual performace distinction through use of colors
- Future workout performance prediction
- Two languages: English and German


## Installation

**To install required packages:**
./installgtk.sh

**To start the program:**
./run.sh
